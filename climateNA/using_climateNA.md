To obtain monthly contemporary and historical climate data at each sampling location, I am using [ClimateNA](https://sites.ualberta.ca/~ahamann/data/climatena.html) version v7.60 (downloaded on April 6th, 2026). 
I opened the software in Windows 11 (virtual machine installed on Mac Sonoma 14.8.5 using [UTM](https://mac.getutm.app/)).
